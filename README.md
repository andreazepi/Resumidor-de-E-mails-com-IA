# Resumidor-de-E-mails-com-IA
Aplicação Python que automatiza o resumo de e-mails usando a API do Google Gemini.  A ferramenta processa múltiplos e-mails, gera resumos concisos em uma linha indicando  o intuito de cada mensagem, e salva os resultados em arquivo. Inclui tratamento de erros,  estatísticas de processamento e controle de taxa de requisições.
