# Resumidor de E-mails com IA

Aplicação que resume e-mails automaticamente usando a API do Google Gemini.

## O que faz?

Processa uma lista de e-mails e gera um resumo de uma linha para cada um, mostrando o intuito da mensagem.

## Como instalar?

1. Clone o repositório:
```bash
git clone https://github.com/seu-usuario/resumidor-emails.git
```

2. Instale as dependências:
```bash
pip install -r requirements.txt
```

## Como usar?

```python
from main import resumidor_de_emails

email_bodies = ["seu e-mail aqui", "outro e-mail aqui"]
resumidor_de_emails(email_bodies)
```

## Requisitos

- Python 3.8+
- Chave API do Google Gemini

- Python
- Google Gemini API
